# Ebola Emergency Support Application

A modern, responsive web application for processing applications for Ebola treatment and emergency financial assistance.

## 🌐 Live Application

**Public URL**: https://ebola-emergency-support-jakemore.zocomputer.io

Access your application from anywhere in the world!

## 🖼️ Real Healthcare Images

We've integrated authentic healthcare worker images from Unsplash:
- ✅ Hero banner with healthcare workers
- ✅ Professional medical team photos  
- ✅ Ebola research and testing imagery
- ✅ Volunteer healthcare workers
- ✅ Doctor treatment scenarios

All images are free to use and properly attributed. See `docs/IMAGE_ATTRIBUTION.md` for details.

## 🌟 Features

### Public Portal
- **Multi-Step Application Form**: 10-step guided application process
- **Progress Tracking**: Visual progress indicator
- **Form Validation**: Real-time validation with Zod
- **Country Selection**: Auto-populated countries with flags
- **Receipt Generation**: Downloadable application summary
- **Mobile Responsive**: Works on all device sizes
